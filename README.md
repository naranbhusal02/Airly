### Installation

1.  **Clone the repository**

    ```bash
    git clone [https://github.com/naranbhusal02/Airly/.git](https://github.com/naranbhusal02/Airly.git)
    cd
    ```

2.  **Open the index file in borwser**

    ```bash
    open index.html
    ```

3.  **File Structure for this project**

```bash
├── logo/               #Logos of radio channels
├── resource/           #Required resources like images/videos
├── comingsoon.css      #CSS Code for coming soon page
├── comingsoon.html     #HTML Structure
├── index.html          #Main home page
├── me.css              #CSS for me.html page
├── me.html             #HTML code for me page
├── readme.me           #Include project description
├── script.js           #main script file
├── styles.css          #main css file
```
